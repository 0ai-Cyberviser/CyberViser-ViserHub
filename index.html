Fuzzing + dataset workflow

Use the public hub as the front door for fuzzing tools and dataset generation.

The site links the public project surface while your Google Admin, Google Drive, and private working files remain private to your account. Use public repos for code and docs; use private storage for raw corpora, unpublished datasets, credentials, crash dumps, and admin records.

04

Promote public docs, release notes, examples, and safe demos to this site while keeping private Drive/Admin assets and sensitive corpora out of GitHub Pages.

Privacy boundary

Public site, private operations.

This domain is designed as a public landing hub. Administrative systems, Google Drive, Google Admin, private repositories, internal notes, and sensitive infrastructure data stay off the public website.

  1. DNS verification: TXT challenge records are public DNS records by design.
  2. Private assets: keep Drive/Admin material restricted to your Google account only.
  3. GitHub Pages: publish only sanitized public-safe content from this repository.
