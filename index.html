Privacy boundary

Public site, private operations.

This domain is a public landing hub. Administrative systems, Google Drive, Google Admin, private repositories, raw telemetry, and sensitive infrastructure data stay off the public website.

  1. Public: GitHub Pages, README links, docs, badges, safe examples, and roadmap.
  2. Private: admin consoles, Drive files, secrets, raw datasets, credentials, and internal notes.
  3. Review-first: all repo improvements should land through PRs, not blind direct edits.
